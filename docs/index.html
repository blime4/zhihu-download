安装指南

1

安装浏览器扩展

先安装 Tampermonkey 扩展：
Chrome 版本 或 Edge 版本

2

安装脚本

点击下方的下载按钮，Tampermonkey 会自动识别并安装脚本

3

开始使用

访问支持的文章页面，点击右上角的下载按钮即可

立即安装脚本

点击下方按钮，Tampermonkey 将自动安装脚本

安装 URL2Markdown 脚本

安装后无需重启浏览器，直接访问文章页面即可使用

使用方法

访问文章页面

打开知乎、CSDN、微信公众号或稀土掘金的文章页面

点击下载按钮

在页面右上角找到 "下载 Markdown" 按钮，点击开始转换

等待处理完成

脚本会自动解析内容、下载图片、生成 ZIP 文件

下载完成

浏览器会自动下载包含 Markdown 和图片的 ZIP 文件
